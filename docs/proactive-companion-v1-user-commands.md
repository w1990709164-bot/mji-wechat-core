# 用户主动消息命令

- `主动消息`：查看当前次数、间隔和免打扰设置
- `主动消息 0`：关闭主动消息
- `主动消息 20`：每天最多 20 次；可填写任意非负整数
- `主动间隔 90分钟`：两次主动消息至少间隔 90 分钟
- `主动间隔 2小时`：两次主动消息至少间隔 2 小时
- `主动间隔 1天`：两次主动消息至少间隔 1 天
- `免打扰 23:30-08:00`：设置个人免打扰时间，可跨午夜
- `关闭免打扰`：取消个人免打扰时间
- `开启免打扰`：恢复默认 23:00—08:00
- `关闭主动`：关闭主动消息
- `开启主动`：按原次数或默认每天 1 次重新开启

用户个人每日次数不设产品固定上限，但仍受管理员全局每日预算、余额和本地相关性筛选限制。
