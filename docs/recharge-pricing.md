# M叽充值计价规则

## 额度与人民币

```text
1 额度 = 0.005 元
```

因此：

```text
10 元 = 2000 额度
30 元 = 6000 额度
50 元 = 10000 额度
```

## 单次正常回复

```text
用户扣除：10 额度 = 0.05 元
模型成本：约 4.5 额度 = 0.0225 元
单次额度差：约 5.5 额度 = 0.0275 元
```

按当前计费：

```text
2000 额度约可完成 200 次正常回复
6000 额度约可完成 600 次正常回复
10000 额度约可完成 1000 次正常回复
```

模型调用失败、请求超时或回复未成功发送时，不应实际扣除用户额度。
