# 付费用户管理后台 v1

## 启动

```powershell
npm run admin:web
```

管理员总后台会继续运行在 `http://127.0.0.1:8787/`，点击顶部“用户管理”后打开 `http://127.0.0.1:8788/`。

也可以单独启动：

```powershell
npm run admin:users-web
```

## 功能

- 用户列表：昵称、微信 ID、状态、可用额度、累计消费、累计充值、待确认订单、最后活跃时间
- 筛选：余额不足、待确认订单、已暂停、已封禁
- 用户详情：人设摘要、长期记忆数量、消息数量、充值订单、额度流水、管理审计
- 人工增加或扣减额度，支持业务编号幂等
- 暂停、恢复、封禁、解封用户
- 设置新用户试用额度，约 30 秒内生效
- 运营概览：总用户数、今日活跃、今日消耗、今日充值金额、待确认订单
