# 新用户如何添加 M叽

M叽微信版当前连接的是一个真实微信账号。新用户不需要安装其他软件，只需要把这个微信账号加为好友。

## 管理员准备

建议使用一个专门的微信账号作为 M叽，不要与私人主账号混用。

在这个微信账号中设置：

- 昵称：M叽；
- 头像：M叽头像；
- 个性签名：可填写“AI陪伴聊天”；
- 微信号：设置成便于搜索的名称；
- 个人二维码：保存为图片，作为主要邀请入口。

## 新用户添加流程

1. 管理员把 M叽微信号、个人二维码或微信名片发给新用户；
2. 新用户在微信中搜索微信号，或扫描个人二维码；
3. 新用户发送好友申请；
4. 管理员在 M叽微信账号上通过好友申请；
5. 新用户给 M叽发送第一条消息，例如“你好”；
6. 程序自动为该微信用户建立独立档案、聊天记录、长期记忆和余额账户；
7. 系统自动赠送 100 试用额度；
8. 第一次回复成功后扣除 10 额度，通常剩余 90。

## 需要保持运行

用户发消息时，运行 M叽的电脑或服务器必须保持：

```powershell
npm start
```

程序停止时，微信账号仍然存在，但 AI 不会自动处理新消息。

## 邀请文案示例

把以下信息和二维码一起发给用户：

```text
这是 M叽的微信账号，扫描二维码添加好友。
通过后直接发“你好”就能开始聊天，新用户会自动获得 100 试用额度。
```

## 注意

- 新用户必须先成为该微信账号的好友；
- 好友申请需要在微信账号上正常通过；
- 同一个微信用户只会建立一个账户；
- 换微信号会被识别为新用户；
- 不要公开数据库、API 密钥或管理员窗口地址；
- 管理员余额窗口只用于你自己管理用户，不是给普通用户使用的页面。
